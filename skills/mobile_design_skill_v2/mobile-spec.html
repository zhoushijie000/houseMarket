基于 SVG 设计稿提炼：375 宽移动端、橙色品牌体系、白底卡片信息流、房产价格与专题标签突出，适用于区县专题、楼盘列表、活动与政策信息页。

三、页面规则

页面宽度按 375px 设计；左右安全边距建议 16px。
卡片内容优先按“标题 → 基础信息 → 价格 → 状态”组织。
价格使用品牌橙色突出，剩余套数使用红色弱底标签强调。
活动与政策模块使用白底卡片承载，时间、地点等信息使用 12px 灰色说明文案。
